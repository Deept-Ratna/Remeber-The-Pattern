How to Play

1. Press any key to start the game.

2. Watch the sequence of colors flashing.

3. Repeat the sequence by clicking on the colored buttons.

4. If you get the sequence correct, the game will go to the next level with an additional color.

5. If you make a mistake, the game is over, and you can restart by pressing any key.
